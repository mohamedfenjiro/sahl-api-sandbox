API Endpoints

Bank Info

Get information about the Sahl API.

GET /bank/info

Create Link Token

Create a link token for client-side initialization.

Comma-separated list: auth, transactions, statements
POST /bank/link/token/create

Exchange Public Token

Exchange a public token for an access token.

POST /bank/item/public_token/exchange

Auth

Get account and routing numbers.

POST /bank/auth/get

Transactions

Get transaction data.

POST /bank/transactions/get

Statements

Get statement data.

POST /bank/statements/get
